## ğŸ¨ Design System Quick Reference

### Colors
- **Primary Blue:** `#1E73BE` (Corefinity brand)
- **Accent Orange:** `#FF7A59` (CTAs, highlights)
- **Success Green:** `#4CAF50` (Performance metrics)
- **Neutral Grays:** `#F9FAFB` to `#111827` (backgrounds to dark text)

### Typography
- **Font Family:** Inter (Google Fonts)
- **Heading Scale:** 36px (mobile) to 56px (desktop) for H1
- **Body Text:** 16px base size
- **Line Height:** 1.625 for body text

### Spacing
- **Base Unit:** 8px
- **Section Spacing:** 96px between major sections
- **Card Padding:** 24px internal padding
- **Container Max Width:** 1280px

### Breakpoints
- **Mobile:** 320px - 767px
- **Tablet:** 768px - 1023px
- **Desktop:** 1024px+
